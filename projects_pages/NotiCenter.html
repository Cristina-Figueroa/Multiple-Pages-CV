NotiCenter - A Campus Event Promoting App

Jan 2024 - May 2024

UPRM NotiCenter is a mobile platform designed to organize and promote campus events for students. It acts as a centralized hub where users can access event information, connect with student associations, and discover recreational activities. Key features include categorized event browsing, personalized notifications, a calendar view, a homepage highlighting upcoming events, and detailed event cards.

Human Computer Interaction · Front-End Development · React Native · Mobile Application Development
